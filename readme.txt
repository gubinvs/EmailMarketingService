Подготовьте HTML-тело письма:

Положите файл email_body.html в папку uploads (или email_body_url.txt с URL, откуда брать HTML).

Отправьте POST-запрос с Excel-файлом (поля с адресами в первом столбце) на POST /api/upload/upload-excel (форма multipart/form-data, поле file).

Фоновый сервис будет отправлять письма пачками по 500 адресов, после каждой пачки — ждать 24 часа (реализовано через метку NextRunUtc в состоянии). После отправки всех — придёт уведомление на gubinvs@gmail.com.
